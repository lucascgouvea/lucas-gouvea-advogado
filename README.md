# ⚖️ Lucas Gouvea Advogado - Website Jurídico Premium

Website jurídico de alto padrão, moderno, 100% responsivo e otimizado para conversão desenvolvido para o escritório **Lucas Gouvea Advogado**. O projeto transmite autoridade, credibilidade, sofisticação e facilita o contato imediato do visitante.

---

## ✨ Recursos Principais

*   **Identidade Visual Premium:** Cores sofisticadas (Azul Escuro `#0F1B2D`, Dourado `#D6AF67`, Off-White `#F5F5F5`), tipografia refinada (Playfair Display & Inter) e espaços em branco generosos.
*   **Modo Escuro (Dark Mode):** Alternância inteligente de tema com salvamento de preferência automática no navegador (`localStorage`).
*   **Otimizado para Conversão:** CTA flutuante do WhatsApp sempre visível, formulário de contato integrado e botões estratégicos de ação rápida.
*   **Animações Suaves:** Transições fluidas ao rolar a página utilizando a biblioteca AOS (Animate on Scroll) e micro-interações nativas via CSS.
*   **SEO & Acessibilidade:** Estrutura HTML5 semântica, metatags Open Graph completas para compartilhamento e dados estruturados (Schema JSON-LD) para otimizar o ranking no Google Maps e buscas locais.
*   **Métricas Animadas:** Contadores dinâmicos na seção de autoridade que ativam de forma automática ao atingir a seção com o scroll.

---

## 🛠️ Tecnologias Utilizadas

*   **HTML5** semântico
*   **CSS3** moderno (CSS Grid, Flexbox e Variáveis de Ambiente)
*   **JavaScript (ES6+)** nativo
*   **AOS (Animate On Scroll)** para animações de rolagem
*   **Lucide Icons** para ícones limpos e minimalistas

## 🚀 Como Executar o Projeto Localmente

Como o projeto utiliza arquivos locais e SVG, é altamente recomendado rodar sob um servidor local estático simples:

1.  Abra seu terminal e acesse a pasta do projeto:
    ```bash
    cd lucas_gouvea_advogado
    ```

2.  Inicie um servidor com Python:
    ```bash
    python3 -m http.server 8000
    ```

3.  Acesse em seu navegador:
    ```text
    http://localhost:8000
    ```
